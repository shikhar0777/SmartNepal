Q-SMART-E

QSMARTE is a nonprofit 8-week summer initiative designed to teach recent high-school graduates research and innovation in various fields. The program focuses on helping future pillars gain early research experience through guided projects, mentorship, and hands-on learning.

This website contains the full description of the program. It is built mainly using TypeScript as the core framework, with a connected database, and deployed on Vercel for scalability and smooth performance.

QSMARTE aims to support students in developing research skills, understanding modern technologies, and preparing for university-level academic work.
